# Guía para Integrar tu Fuente

## Paso 1: Extraer los archivos
1. Extrae el ZIP con tu fuente
2. Copia los archivos de fuente (.ttf, .otf, .woff, .woff2) a:
   ```
   assets/fonts/mi-fuente/
   ```

## Paso 2: Información necesaria
Necesito saber:
- **Nombre de la fuente** (ej: "Inter", "Montserrat")
- **Archivos que tienes** (ej: "Inter-Regular.ttf", "Inter-Bold.ttf")
- **Pesos de fuente** (Regular=400, Medium=500, Bold=700, etc.)

## Paso 3: Ejemplo de estructura
Si tu fuente se llama "Inter", los archivos deberían estar así:
```
assets/fonts/inter/
  - Inter-Regular.ttf
  - Inter-Medium.ttf
  - Inter-Bold.ttf
```

## Paso 4: Una vez que tengas los archivos
Dime:
1. El nombre de la fuente
2. Qué archivos tienes
3. Y yo te ayudo a integrarla completamente
